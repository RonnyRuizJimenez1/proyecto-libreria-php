# 📚 Proyecto Final - Librería Online

## 🌐 Demo en vivo

👉 https://libreria-ronny.rf.gd

---

## 🚀 Descripción

Sistema web desarrollado con PHP, MySQL, HTML, CSS y Bootstrap que permite:

* 📖 Ver listado de libros
* ✍️ Consultar autores
* 📩 Enviar mensajes de contacto
* 📲 Integración directa con WhatsApp

---

## 🛠️ Tecnologías utilizadas

* PHP (PDO)
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

---

## 📂 Estructura del proyecto

* `index.php` → Página principal
* `libros.php` → Catálogo de libros
* `autores.php` → Lista de autores
* `contacto.php` → Formulario + WhatsApp
* `config.php` → Conexión a base de datos

---

## 🧠 Funcionalidades

* 🔍 Búsqueda de libros
* 📱 Diseño responsive
* 🔐 Conexión segura con PDO
* 📲 Envío automático a WhatsApp
* 💾 Almacenamiento en base de datos
